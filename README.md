# BNK Wellness

A modern, responsive e-commerce website for wellness products.

## Tech Stack
- HTML5
- CSS3 (Custom properties, Flexbox, Grid, Animations)
- JavaScript (Vanilla ES6+)

## Features
- **Responsive Design:** Works on mobile, tablet, and desktop.
- **Product Catalog:** Categorized wellness products.
- **Product Details:** Detailed views with images, descriptions, and FAQs.
- **Shopping Cart:** Add to cart functionality with LocalStorage persistence.
- **Checkout Flow:** Address collection form (Payment gateway simulated).
- **Fluid Animations:** Smooth transitions and interactions.

## How to Run
Simply open `index.html` in your web browser.

## Structure
- `css/`: Stylesheets
- `js/`: JavaScript logic and data
- `index.html`: Homepage
- `product-detail.html`: Individual product page
- `cart.html`: Cart and checkout page
